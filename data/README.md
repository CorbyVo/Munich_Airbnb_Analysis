# Data

This project uses public Airbnb listing data from Inside Airbnb.

Download page:

```text
https://insideairbnb.com/get-the-data/
```

City:

```text
Munich, Bavaria, Germany
```

Recommended file:

```text
listings.csv
```

Put the file here:

```text
data/raw/listings.csv
```

If you download the compressed detailed file, keep it as:

```text
data/raw/listings.csv.gz
```

The analysis script supports both file names.
